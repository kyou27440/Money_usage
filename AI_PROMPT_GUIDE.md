## 📝 모듈 A: 개인 가계부 프롬프트

### A-1. 수입/지출 입력

```
개인 가계부에 아래 내역을 입력해줘:
- 날짜: 2026-07-21
- 구분: 지출
- 카테고리: 식비
- 금액: 250,000 VND
- 메모: 점심 식사 (회사 근처 쌀국수)
```

### A-2. 여러 건 일괄 입력

```
개인 가계부에 아래 내역들을 일괄 입력해줘:

| 날짜 | 구분 | 카테고리 | 금액(VND) | 메모 |
|------|------|----------|-----------|------|
| 2026-07-21 | 지출 | 식비 | 250,000 | 점심 |
| 2026-07-21 | 지출 | 교통비 | 50,000 | 그랩 택시 |
| 2026-07-21 | 입금 | 급여 | 30,000,000 | 7월 급여 |
```

### A-3. 내역 조회

```
개인 가계부에서 이번 달(2026년 7월) 지출 내역을 카테고리별로 합산해서 보여줘.
```

```
개인 가계부에서 최근 1주일간 수입/지출 내역과 잔액을 보여줘.
```

### A-4. 카테고리 관리

```
개인 가계부에 새 카테고리를 추가해줘:
- 카테고리명: 의료비
- 구분: 지출
- 아이콘: 🏥
```

## ⛳ 모듈 B: 회사 모임 관리 프롬프트

### B-1. 게임 기록 입력

```

- 참여자 및 순위:
  1등: 김철수
  2등: 이영희
  3등: 박민수
  4등: 정대호

```

```
- 합류일: 2026-07-21

```


```
멤버 '최준호'를 퇴사(departed) 처리해줘. 이탈일: 2026-07-25.
```

### B-3. 회비 관리

```

- 박민수: 200,000 VND 입금

```


### B-4. 조회

```
모임 멤버별 현재 회비 잔액을 보여줘.
```

```
최근 5회 게임에서 멤버별 등수 변동 추이를 그래프로 보여줘.
```

